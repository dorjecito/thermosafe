# ThermoSafe – Risc per calor 🌞

Aplicació web que mostra el risc per calor en funció de la temperatura i humitat actuals.  

📍 Basat en la ubicació GPS o cerca manual per ciutat.

### Funcionalitats
- Índex de calor percebut (heat index)
- Alertes visuals i sonores si el risc és alt
- Consulta recurrent automàtica cada 30 minuts
- Selecció de ciutats habituals de les Illes Balears
- Interfície moderna i multilingüe (català/castellà)

### Tecnologies
- React + Vite
- TypeScript
- OpenWeather API
- Notification API
